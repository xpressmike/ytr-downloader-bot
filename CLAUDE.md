## Шаблон подписи (caption)

Собирай подпись СТРОГО по этому шаблону (HTML, каждая строка = новая строка):

🎬 <b>{title}</b>

📝 {summary}

👤 <b>Автор:</b> {uploader}
⏱ <b>Длительность:</b> {duration_formatted}
📱 <b>Источник:</b> {platform_emoji} {platform_name}
🏷 <b>Категория:</b> #{category}

{tags через пробел}

<a href="{original_url}">🔗 Смотреть оригинал</a>

## Правила форматирования полей

**HTML-эскейп** (обязательно для title, uploader, summary, и любого другого
пользовательского текста, попадающего в шаблон):
- `&` → `&amp;`
- `<` → `&lt;`
- `>` → `&gt;`
- эскейпь ДО оборачивания в `<b>...</b>` или другие теги шаблона

**title** — используй title из метаданных yt-dlp как есть,
но если он начинается с "Video by ..." или содержит только имя автора
без сути — замени его на краткий заголовок, который ты сам придумаешь
по транскрипту (3-7 слов, отражающих содержание).
Если итоговый title > 150 символов — обрежь до 147 и добавь `…`.

**summary** — 2-3 конкретных предложения на русском о содержании видео.
Не пиши "в этом видео", "автор показывает" — сразу по сути.

**uploader** — имя автора из метаданных. Если поле пустое — пиши "—".

**duration_formatted** — форматируй длительность:
- меньше 60 сек → "15 сек"
- от 60 сек до часа → "2:35"
- больше часа → "1:15:20"

**platform_emoji + platform_name** — определяй по original_url:
- youtube.com/shorts, youtu.be, youtube.com → 📺 YouTube
- instagram.com/reel, instagram.com/p → 📸 Instagram
- tiktok.com → 🎵 TikTok
- иначе → 🌐 Web

**category** — одно слово из: обучение / юмор / новости / рецепт /
фитнес / техника / музыка / путешествия / спорт / животные / лайфхак / другое

**tags** — 5-10 конкретных поисковых тегов в формате #тег:
- только на русском, нижний регистр
- без пробелов (используй _ : #паста_карбонара, #машинное_обучение)
- конкретные существительные (#декораторы, #swift — не #программирование)
- если упомянут продукт/инструмент/место — отдельным тегом
- ОБЯЗАТЕЛЬНО добавь тег языка: #ru или #en (в зависимости от языка речи)

## Что НЕ делать

- НЕ копируй description из yt-dlp метаданных в подпись — там мусор
  типа "Video by ... Follow for more..." и рекламные хэштеги автора
- НЕ используй английские слова для полей (Автор, не Author)
- НЕ добавляй строки, которых нет в шаблоне
- НЕ обрезай теги — они критичны для поиска в Telegram
- Если итоговая подпись > 1024 символов — сокращай ТОЛЬКО summary,
  шапку и теги не трогай
